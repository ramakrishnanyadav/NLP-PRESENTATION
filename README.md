# AI for Data Analysis and Visualization in Research

## Project Overview
This project is an interactive academic presentation on using Artificial Intelligence for Data Analysis and Visualization. It was developed to explore how AI techniques compare with traditional methods, highlight modern AI tools, and provide a visually engaging, modern presentation format.

## Development Setup

To run this project locally, ensure you have Node.js installed, then follow these steps:

1. **Clone the repository:**
   ```sh
   git clone https://github.com/ramakrishnanyadav/NLP-PRESENTATION.git
   cd NLP-PRESENTATION
   ```

2. **Install dependencies:**
   ```sh
   npm install
   ```

3. **Start the development server:**
   ```sh
   npm run dev
   ```

## Technology Stack
- **Framework:** React + Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS + Custom Dark Mode
- **Animations:** Framer Motion & HTML5 Canvas (Particle physics)
- **UI Components:** shadcn/ui
